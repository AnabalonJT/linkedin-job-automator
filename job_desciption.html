About the Role

We are seeking a highly technical, systems-minded FullStack Shopify Developer to own and scale a diverse portfolio of e-commerce and content-driven web properties.

This is not just a development role — it is an architectural position. You will manage infrastructure, eliminate technical debt, optimize performance, and build custom integrations that power both D2C and wholesale operations. You will operate as a “department of one,” with full ownership over deployment, performance, integrations, and long-term scalability.

Key Responsibilities:

Portfolio & Infrastructure Management

  • Maintain, audit, and strategically upgrade 10+ web properties across:
  • Custom WordPress/PHP builds
  • Shopify (D2C)
  • WooCommerce (Wholesale)
  • Own Git-based deployment workflows across all domains, ensuring:
  • 99.9% uptime
  • Robust backup protocols
  • Proactive security patching
  • Systematically eliminate technical debt and unify deployment strategy
  • Manage DNS, hosting, and domain infrastructure stability

Performance & Conversion Optimization

  • Improve Core Web Vitals and drastically reduce page load times
  • Optimize sites for conversion rate and user journey flow
  • Implement advanced tracking via Google Tag Manager and custom events
  • Stay current with trends in readability, UX, SEO, and AEO (AI-Enhanced Optimization)

Custom Development & Design Execution

  • Build high-converting custom landing pages
  • Develop custom WordPress themes and plugins (not dependent on page builders)
  • Implement custom Shopify storefront enhancements
  • Lead technical execution of full website redesigns

Systems Integration & Data Synchronization

  • Architect and maintain API integrations between:
  • Shopify / WooCommerce
  • NetSuite (Inventory & ERP)
  • CRMs
  • Operational tools
  • Build and maintain webhooks and middleware automations
  • Reduce manual overhead through intelligent workflow automation
  • Ensure real-time data accuracy across systems

E-Commerce Architecture

  • Configure complex WooCommerce wholesale environments (bulk pricing, workflows)
  • Build and maintain high-performance Shopify D2C environments
  • Support headless Shopify implementations when applicable
  • Ensure seamless inventory and fulfillment integrations

Technical SEO & Analytics

  • Implement advanced GTM setups with custom conversion tracking
  • Optimize site architecture for crawlability and schema markup
  • Analyze performance and analytics data to inform development priorities

Collaboration & Leadership

  • Partner with marketing, content, and design teams
  • Provide technical feasibility guidance for campaigns
  • Participate in team meetings and respond to urgent technical issues
  • Mentor junior developers as the team grows
